Aquí te presento un ejemplo de código que muestra los números pares hasta el 100: javascript for (let i = 2; i <= 100; i += 2) { console.log(i); } Este programa utiliza un bucle for para iterar desde 2 hasta 100, aumentando de 2 en 2 en cada iteración. Dentro del bucle, se utiliza la función console.log() para mostrar el número actual en la consola del navegador. Como solo se muestran los números pares, solo se muestran los números que son divisibles por 2. También puedes utilizar la misma lógica con un bucle while: javascript let i = 2; while (i <= 100) { console.log(i); i += 2; } Este programa utiliza un bucle while para iterar desde 2 hasta 100, aumentando de 2 en 2 en cada iteración. Dentro del bucle, se utiliza la función console.log() para mostrar el número actual en la consola del navegador. Como solo se muestran los números pares, solo se muestran los números que son divisibles por 2.
